Docker Webserver Tutorial
We will be following the steps from the link below to get Docker setup with a Flask webserver!

Prereqs: Make sure you have installed both Docker Engine and Docker Compose

Step 1: Make a directory on your local machine for this project.

Step 2: Create a file called "app.py" and copy the given Python code from the link below in this step.

Step 3: Create a file called "requirements.txt" and copy and paste the given code from the step on the webpage. This contains all the needed Python libraries that will be needed to run the program.

Step 4: Create a file called "Dockerfile" with no extension and copy the code from the site.

Step 5: Create a file called "docker-compose.yml" and copy the code from the site.

Step 6: Start up your application with the command called "docker-compose up".

Step 7: Go to "http://localhost:5000/" in your browser to see your live app running! The container as executed this flask file which is a webserver that on the index page in localhost on this port, displays the method called hello in the Python code. This flask server can also be used to render other webpages that you develop locally and can be displayed on this locally hosted server.
