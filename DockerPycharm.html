Docker with Pycharm Tutorial
The following steps shows how to connect Docker with Pycharm to write python code.!

Install and run Docker and make sure that the Docker plugin is enabled.

Step 1: Create a project and add a Python script.
Create a Python project QuadraticEquation, add the Solver.py file and enter the code provided

Step 2: Configure Docker as a remote interpreter.

Step 3: Run application in a Docker container,
In the left gutter, next to the main clause, click the Run button, and choose Run 'Solver.py' command. You see that your script runs in the Docker container:

Step 4: Debug application in Docker container.
For that, let's put a breakpoint on the line that calculates d, then click Run and choose Debug 'Solver'.

Step 5: Use Docker tool window
Connect to a Docker daemon Run tool window, we'll see that the Docker prefix corresponds to the container ID in the Properties tab of the Docker tool window Open the Debug tool window, you will see that the Docker prefix (another one!) corresponds to the another container ID in the Properties tab of the Docker tool window: Finally, you will see the strange names of the containers - they are human-readable and generated by Docker.
